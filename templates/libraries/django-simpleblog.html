django-simpleblog 1.0 (python, django)

Simple blog app for Django project.

http://pypi.python.org/pypi/django-simpleblog
https://bitbucket.org/sikaondrej/django-simpleblog/overview

Installation

You have 2 way to install django-simpleblog. I prefer install via python package index

pip install django-simpleblog

or download django-simpleblog-1.0.tar.gz and install manually

wget http://pypi.python.org/packages/source/d/django-simpleblog/django-simpleblog-1.0.tar.gz
tar xzvf django-simpleblog-1.0.tar.gz
cd django-simpleblog-1.0
python setup.py install

Example django projdect

Example project is for Django 1.4 and is locatei in example dir in GIT repository. 
$ git clone https://bitbucket.org/sikaondrej/django-simpleblog.git
$ cd django-simpleblog/example
$ python manage.py syncdb
$ python manage.py runserver
