viminput 1.0.1 (python)

If you want to enter lot of text, python's standart input is inappropriate. You write data comfortably in vim, but python load data as data from std input.

http://pypi.python.org/pypi/viminput
https://bitbucket.org/sikaondrej/python-viminput/overview

Installation

You have 2 way to install viminput. I prefer install via python package index

pip install viminput

or download viminput-1.0.1.tar.gz and install manually

wget http://pypi.python.org/packages/source/v/viminput/viminput-1.0.1.tar.gz
tar xzvf viminput-1.0.1.tar.gz
cd viminput-1.0.1
python setup.py install

Example

import functions

>>> from viminput import viminput, textinput

viminput without default data

open blank vim

>>> data = viminput()

viminput with default data

open vim with default data

>>> data = viminput("this is default data\ndataon new line")

textinput open editor from $EDITOR with default data

open texteditor from $EDITOR with default data

>>> data = textinput(content="this is default data\ndataon new line")

textinput open defined editor

open texteditor (nano) with default data

>>> data = textinput(editor="nano", content="this is default data\ndataon new line")
