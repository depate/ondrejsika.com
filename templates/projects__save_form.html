SaveForm

O projektu

Aplikace sloužící pro sběr dat z uživatelsky vytvořených formulářů bez potřeby psaní jakéhokoliv kódu na spracování dat.

Dokumentace

API

Pro záznam dat používá "saveform" jednoduché API. Jedná se o HTTP GET request na url /api/add. Parametr __form_pk je povyný a udává, k jakému formuláři se mají dat zapisovat. Volitelný parametr __response ukazuje kam se má přesměrovat stránka po zápisu do db. Většinou jde o vaší "thankyou" stránku. Příklad url: http://localhost:8000/api/add/?__form_pk=1&a=1&b=2&c=3.

Pro výsledky se zobrazují na url /form-[form_pk]. Příklad url: http://localhost:8000/form-1/.
