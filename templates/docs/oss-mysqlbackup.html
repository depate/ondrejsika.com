Tool for backup MySQL database

Authors

Ondrej Sika ondrejsika.com dev@ondrejsika.com

Source

Documentation http://ondrejsika.com/docs/oss-mysqlbackup
Bitbucket https://bitbucket.org/sikaondrej/oss-mysqlbackup

Usage

$ oss-mysqlbackup [mysql_user] [passwd] [/path/to/export]            # save sql files
$ sudo oss-mysqlbackup-cron [mysql_user] [passwd] [/path/to/export]  # add oss-mysqlbackup to cron (once per day)
