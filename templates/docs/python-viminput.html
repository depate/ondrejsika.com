viminput documentation

If you want to enter lot of text, python's standart input is inappropriate. You write data comfortably in vim or $EDITOR, but python load data as data from std input.

Authors

Ondrej Sika ondrejsika.com dev@ondrejsika.com

Source

Documentation http://ondrejsika.com/docs/python-viminput
Python Package Index http://pypi.python.org/pypi/viminput
Bitbucket https://bitbucket.org/sikaondrej/python-viminput

Instalation

Instalation is very simple via pip. 
# pip install viminput

Usage

from viminput import viminput, textinput
# use vim
text = viminput()
print text
# use $EDITOR
text = textinput()
# set editor
text = textinput(editor="nano")
# default data
text = viminput(content="default data")
