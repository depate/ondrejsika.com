Installation

git clone git://github.com/sikaondrej/spm.git
cd spm
./setup install

Usage

if you aren't root or don't use sudo, you must type password for user postgres.

Run psql

spm

Run sql command

spm pqsl "sql command"

Create db

spm createdb "database"

Create user

must be root 
spm createuser "user"

Grantall 

spm grantall "user" "database"

help

spm help
