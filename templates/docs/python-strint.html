Instalation

Instalation is very simple via pip. 
# pip install strint

Functions

encode

from strint import encode, decode
>>> encode(1000)
'g8'
>>> encode(1000, "abc")
'bbabaab'

decode

>>> decode("g8")
1000
>>> decode("bbabaab", "abc")
1000
