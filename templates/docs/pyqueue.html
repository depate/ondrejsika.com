Installation

simple via pip 
# pip install pyqueue

Usage

>>> from pyqueue import Queue
>>> queue = Queue()
>>> queue.empty()
True
>>> queue.put(1)
>>> queue.empty()
False
>>> queue.put(2)
>>> queue.get()
1
>>> queue.empty()
False
>>> queue.get()
2
>>> queue.empty()
True
