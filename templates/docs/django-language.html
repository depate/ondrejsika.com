Instalation

Instalation is very simple over pip. 
# pip install django-language

settins.py

INSTALLED_APPS += ("django_language", )
TEMPLATE_CONTEXT_PROCESSORS += ("django_language.context_processors.current_language", )

in template

<p>
    current language code: {{CURRENT_LANGUAGE}}
<br>
    current language name: {{CURRENT_LANGUAGE_NAME}}
