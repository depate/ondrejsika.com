Instalation

Instalation is very simple via pip. 
# pip install normpath

Usage

user@host:~/python $ python 
>>> from normpath import normpath
>>> normpath("..", "apps/myapp")
"/home/user/apps/myapp"
file /home/user/python/test.py 
from normpath import normpath
print normpath(__file__)
print normpath(__file__, "..")
