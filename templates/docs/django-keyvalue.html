Instalation

Instalation is very simple over pip. 
# pip install django-keyvalue

Usage

settings.py

INSTALLED_APPS += ("keyvalue", )
KEYVALUE_KEYS = (
    ("key1", "default value"),
    ("key2", ""),
)

Keys must be unique

Get value

>>> from keyvalue.models import Storage
>>> Storage.onjects.get(key="key1").value
"default value"
