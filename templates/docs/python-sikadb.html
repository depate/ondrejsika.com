sikadb documentation

Python DB models (like Django models) based on MongoDB

Authors

Ondrej Sika ondrejsika.com dev@ondrejsika.com

Source

Documentation http://ondrejsika.com/docs/python-sikadb
Python Package Index http://pypi.python.org/pypi/sikadb
Bitbucket https://bitbucket.org/sikaondrej/python-sikadb

Instalation

Instalation is very simple via pip. 
# pip install sikadb

Usage

Open database

import sikadb

dbname = "my_db"
db = sikadb.Database(dbname)

Create model

import sikadb

class MyModel(sikadb.Model):
    fields = ["name", "text", "array"]
    database = db

Create and save object

a = MyModel().new(name="object_a")
a.text = "This is object a"
a.array = [1,2,3]
a.save()

Get one object and update

a = MyModel().get(name="object_a")
a.array += [4,5,6]
a.save()

Get more objects

b = MyModel().new(name="b", text="Hello").save()
c = MyModel().new(name="c", text="Hello").save()

MyModel().filter(text="Hello")  # return list of model b, c

Delete object

a = MyModel().get(name="object_a")
a.delete()
