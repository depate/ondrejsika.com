Instalation

Instalation is very simple over git. 
# git clone https://github.com/sikaondrej/alsa-remote.git /opt/alsa-remote

Nginx

server {
    listen 80;
    server_name alsa.localhost;

    location / {
        proxy_pass http://localhost:11220/;
    }
}

Supervisor

[program:alsaremote]
command = gunicorn wsgi:application -b 127.0.0.1:11220
directory = /opt/alsa-remote/
