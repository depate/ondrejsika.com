Instalation

Install script

Instalation is very easy via install-scripts 
wget https://raw.github.com/sikaondrej/install-scripts/master/github-shop-api-install.sh
sudo sh alsa-remote-install.sh
Now you have access to api at localhost:11230.

Manually

Clone repo to app dir. In root are 2 files (app.py, wsgi.py). You can run in Flask server 
python app.py
or 
./app.py
in development mode or in priduction mode via gunicorn (or uwsgi, ...) 
gunicorn wsgi:application -b 0.0.0.0:11230
or 
gunicorn app:app -b 0.0.0.0:11230
