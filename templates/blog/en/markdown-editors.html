I would like to introduce you to some markdown editors, which according to me, it's good fo knowing

Light weight web editor

This web markown editor is pretty for its simplicity. Author: James Taylor. Source on GitHub. Special feature is that, markdown file (encoded by base64) saves to url after "#"

example: http://jbt.github.io/markdown-editor/

Native markdown editor for MAC

It's very similar to jbt's markdown editor, but this is native for MAC. Project is on mouapp.com.

Web extended editor (IDE)

Markable.in is good markdown IDE. You may have account and manage your files online. Disadvantage is low speed of pretty preview.

Web editor works with GitHub, Google Drive, Dropbox

This editor Dillinger, can be connect with your data host and editing markdown files remote (on web), with are synce by Dropbox to local directories.
