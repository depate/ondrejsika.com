Linux

install via pip 
sudo pip install massresize
or 
wget https://pypi.python.org/packages/source/m/massresize/massresize-1.0.0.tar.gz
tar xzvf massresize-1.0.0.tar.gz
cd massresize-1.0.0
sudo python setup.py install
and run from console 
massresize [dir with photos] [width]
example 
massresize /home/sika/Photos/ 300
or in gui massupload-gui

Windows

Download python 2.7 for windows from http://www.python.org/ftp/python/2.7.5/python-2.7.5.msi and pil http://effbot.org/downloads/PIL-1.1.7.win32-py2.7.exe
Download and unzip https://github.com/sikaondrej/massresize/archive/master.zip
