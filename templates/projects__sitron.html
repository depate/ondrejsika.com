Sitron project

O projektu

Projekt Sitron byl vytvořen, protože není žádný pořádjný nástroj, pro pořádání závodů v Orientačním běhu, který funguje na Linuxu.

Tento projekt je OpenSource, šířen pod licencí GNU/GPL v.3.

Dokumentace

URL API

Androidí / desktopová aplikace pošle http požadavek na server kde beží sitron (sitron.app.ondrejsika.com) a on zpracuje přijatá data a vrátí odpověď.

Záznam z rádiokontroly

Sitroin naslouchá na url /api/save_si_data/, všechny požadavky na tuto url se snaží spracovat jako nové ražení radiokontroly. Data jsou posílína v GET parametrech:

  • si - číslo čipu
  • checkpoint_id - kód kontroly
  • race_pk - id závodu

Pokud vše proběhne v pořádku, server odpoví "OK", pokud ne, odpoví "err".
Příklad url requestu je /api/save_si_data/?si=236993&race_pk=1&checkpoint_id=100
